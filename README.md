# PDF 工坊

基于 **pywebview** 的现代化桌面 PDF 工具，提供三大功能：

| 功能 | 说明 | 输出（保存到「下载」目录） |
|------|------|------|
| **解除限制** | 用 `pikepdf` 移除复制/打印/编辑等权限限制与加密 | `原文件名-decrypted.pdf` |
| **合并** | 拖入多个 PDF，拖动调整顺序后合并 | `首个文件名-merged.pdf` |
| **拆分** | 选择一个 PDF，左侧预览，右侧填写多段页码范围和可选文件名 | `文件名.pdf` 或 `原文件名-页码范围.pdf` |

## 安装

```bash
pip install pywebview pikepdf pypdf
```

> pywebview 在不同系统下需要对应的 GUI 后端：
> - **Windows**：自带 EdgeWebView2（Win10/11 已内置），无需额外安装
> - **macOS**：自带 WebKit，无需额外安装
> - **Linux**：`pip install pywebview[qt]` 或 `pip install pywebview[gtk]`（并安装相应系统库）

## 运行

```bash
python app.py
```

调试模式（打开开发者工具）：`python app.py --debug`

## 使用要点

**解除限制**
拖入任意数量的 PDF 后点「解除限制」。若 PDF 设有*打开密码*，在下方填写后再处理；仅有*权限限制*（无打开密码）的文件可直接解除。

**合并**
拖入 ≥2 个文件，用每行左侧的 ⠿ 手柄拖动排序，点「合并」。原文件的书签会一并并入。

**拆分**
只能选择一个 PDF。选择后左侧显示 PDF 预览，右侧显示拆分表格。
- 默认第一行是 `1-2`，最后永远保留一个空白行；在空白行输入数字或文件名后会自动新增下一空白行。
- 范围写法：`1-5, 8, 10-12`（逗号分隔，支持单页与区间），每条范围生成一个文件，填写顺序可以任意。
- 文件名：填写后输出为该文件名；不填写时输出为 `原文件名-页码范围.pdf`。
- **覆盖原标签**复选框：勾选则丢弃源文件原有书签；默认不勾选，保留原书签。

所有输出文件若重名会自动追加 ` (1)`、` (2)` 以避免覆盖。

## 查看错误 / 日志

程序运行时会把所有信息写进 `logs/pdf_workshop.log`（自动滚动，保留最近几份），包括 Python 异常的完整堆栈、未捕获异常、以及前端 JS 报错。看报错有三种方式：

1. **界面右上角「日志」按钮** —— 弹出日志查看窗，可「刷新」「复制全部」，或「打开文件夹」直接定位到日志文件。
2. **失败的结果行** —— 直接点该行即可复制错误信息（行尾也有「复制」按钮）。
3. **从终端启动** —— 错误同时打印到终端；调试前端用 `python app.py --debug` 打开开发者工具。

日志文件路径：程序目录下的 `logs/pdf_workshop.log`（若该目录只读，则退回到「下载」目录）。

## v0.1i 更新

- 拆分页仅允许选择一个 PDF；选择后隐藏大拖放框，只显示“重新选择”。
- 拆分页改为左右结构：左侧 PDF 预览，右侧拆分表格。
- 拆分表格默认显示 `1-2` 和一个空白行；输入后自动增加下一空白行，删空倒数第 2 行时自动删除，始终只保留一个末尾空行。
- “书签名”改为“文件名”：填写后按该文件名输出；不填写则使用 `原文件名-页码范围.pdf`。
